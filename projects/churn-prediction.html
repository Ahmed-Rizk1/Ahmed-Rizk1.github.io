Bank Customer Churn Prediction

Machine Learning MLOps Classification

Project Overview

This project is a comprehensive guide to building a production-grade Churn Prediction system. It goes beyond simple modeling to include robust MLOps practices, including experiment tracking with MLflow, model versioning, and deployment via FastAPI.

Key Features

  • XGBoost & Random Forest models for high-accuracy classification.
  • Automated experiment tracking using MLflow.
  • Dockerized API for easy deployment.
  • CLI interface for training and inference management.

Methodology

The dataset was preprocessed with careful feature engineering (handling categorical variables, scaling). I used SMOTE to handle class imbalance. The best performing model was packaged into a FastAPI application, allowing for real-time churn probability predictions via REST API endpoints.

Technology Stack

XGBoost Scikit-learn MLflow FastAPI Docker
