Land Classification (EuroSAT)

Computer Vision Deep Learning TensorFlow

Project Overview

This project targets environmental monitoring by automating the classification of land use from satellite imagery. Using the EuroSAT dataset and Sentinel-2 satellite images, I developed a robust deep learning pipeline capable of distinguishing between 10 different land-use types (e.g., Forest, Highway, Industrial).

Key Features

  • Custom CNN architecture optimized for satellite imagery.
  • Data augmentation pipeline to handle class imbalance and improve generalization.
  • Comparitive analysis with Transfer Learning (ResNet50, VGG16).
  • Achieved 94% accuracy on held-out test set.

Methodology

The core challenge was extracting features from lower-resolution satellite data. I employed deep Convolutional Neural Networks (CNNs) and experimented with state-of-the-art architectures. Preprocessing involved normalization and specific augmentations (rotation, flip) relevant to aerial views.

Technology Stack
