YOLOv8 Object Detection App

Computer Vision YOLOv8 Streamlit

Project Overview

A powerful real-time object detection application leveraging the YOLOv8 architecture. This app allows users to upload images or videos and perform object detection, tracking, and specific region analysis (ROI).

Key Features

  • Integration of YOLOv8 for state-of-the-art detection speed and accuracy.
  • Region of Interest (ROI) selection to focus detection on specific areas.
  • Object tracking capabilities for video feeds.
  • Dashboard analytics for detection counts and confidence scores.

Methodology

The app wraps the Ultralytics YOLOv8 model in a Streamlit interface. I implemented custom drawing logic with OpenCV to handle ROI selection and tracking logic to maintain object identity across frames.

Technology Stack

YOLOv8 Streamlit OpenCV Python
