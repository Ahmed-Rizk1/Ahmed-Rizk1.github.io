DistilBERT IMDb Finetuning

NLP Deep Learning Transfer Learning

Project Overview

This project focuses on fine-tuning the DistilBERT model on the IMDb dataset for sentiment analysis. The goal was to achieve high accuracy while maintaining specialized efficiency suitable for deployment. I implemented advanced techniques like LoRA (Low-Rank Adaptation) and quantization to optimize the model.

Key Features

  • Fine-tuned `distilbert-base-uncased` on 50k movie reviews.
  • Implemented comprehensive evaluation metrics (Accuracy, F1-Score).
  • Modular codebase with clear separation of training, evaluation, and inference.
  • Interactive inference pipeline via Streamlit UI.

Methodology

The training pipeline uses the Hugging Face `Trainer` API. I experimented with different learning rates and batch sizes to find the optimal hyperparameters. To reduce the model footprint, 8-bit quantization was applied during inference, significantly speeding up prediction times without major accuracy loss.
