Car Price Predictor

Machine Learning Streamlit Regression

Project Overview

An interactive web application helping users estimate the market value of used cars. This project demonstrates the full lifecycle of an ML project from data cleaning and exploratory data analysis (EDA) to model training and deployment.

Key Features

  • Interactive User Interface built with Streamlit.
  • Real-time prediction based on user inputs (Year, Brand, Mileage, etc.).
  • Comparison of multiple regression models (Lasso, Ridge, Random Forest).
  • Production-ready code structure with proper configuration management.

Methodology

The dataset underwent rigorous cleaning to handle missing values and outliers. I utilized feature encoding for categorical variables. The application uses a saved model pipeline to perform inference on new data points provided by the user interface.

Technology Stack

Streamlit Scikit-learn Pandas Joblib
